# Garage Log — プロダクト仕様（SPEC）

最終更新: 2026-07-24  
対象: **全車種**（OBD-II 対応車）  
対象読者: 自分 / Cursor（実装時は本ファイルを正とする）

---

## 1. ひとことで言うと

**どんな車でも使える、Webの整備手帳＋簡易OBD診断。**

「次に何をすればいいか」が一目で分かることを売りにする。

---

## 2. コア感情

ユーザーが最初の約30秒で感じるべきこと:

> 「自分の車の次の整備が、すぐ分かった」

判定:

- その機能は「次の整備・いまの車の状態」を分かりやすくするか？ → **残す**
- 特定メーカー専用・診断チューナー専門ツールに見えるか？ → **切る / 後回し**

---

## 3. 想定ユーザー

- 1台以上の車を自分で世話するオーナー
- メーカー・車種は問わない（国産・輸入・軽・EV）
- 技術に詳しくなくても使える
- Web（Chrome / Edge）→ 将来 Mac でアプリ化・販売

---

## 4. やる / やらない

### やる（いまの幹）

- **複数車両**の登録・切替（無料版は上限あり）
- 車種テンプレートから整備メニュー初期化（NA / ターボ / ディーゼル / HV / EV / 軽 / 最小）
- 走行距離・月数ベースのリマインド
- 整備実施記録（日付・距離・費用・メモ）
- Web Bluetooth（BLE ELM327）による OBD-II 簡易診断（メーカー非依存）
- DTC 読取・消去（確認付き）
- **メーカー別 DTC 辞書ルーティング**（OEM JSON 遅延読込 + 汎用フォールバック + 重要度表示）
- 任意 Gemini による参考アドバイス
- 端末内 `localStorage` 保存
- 旧 Giulietta 専用データからの自動移行

### やってもよい（販売に向けた後続）

- 有料プラン（台数無制限・エクスポート・クラウド同期）
- CSV / PDF エクスポート
- ストア公開（Flutter 等）
- ユーザー自身の API キー不要なサーバ構成
- 利用規約・プライバシーポリシー・免責の整備

### やらない

- 工場級診断ツールの完全代替を主張
- 特定車種だけのハードコード（例: 1車種専用メニューのみ）
- クラシック Bluetooth（SPP）のみアダプタのブラウザ直結
- 過大な故障断定表現

---

## 5. 車両テンプレート

| ID | 用途 |
|----|------|
| gasoline-na | ガソリン自然吸気 |
| gasoline-turbo | ガソリンターボ |
| diesel | ディーゼル（DPF含む） |
| hybrid | HV / PHEV |
| ev | 電気自動車 |
| kei | 軽自動車 |
| minimal | オイル＋車検のみ |

間隔は**目安**。取扱説明書・整備記録を正とし、車両ごとに編集可能。

---

## 6. 画面構成

1. **ホーム** — 車両選択・次の整備・距離更新
2. **整備** — メニュー・記録・履歴
3. **OBD** — 接続 / デモ / メーター / DTC
4. **設定** — 車両追加・編集・テンプレ適用・APIキー

---


## 8. 販売・マネタイズ方針（案）

| 区分 | 内容 |
|------|------|
| 無料 | 車両5台まで・基本整備記録・OBD・ローカル保存 |
| Pro（将来） | 台数無制限・履歴エクスポート・クラウド同期・広告なし |

※ 決済・認証は未実装。定数 `FREE_VEHICLE_LIMIT` で上限のみ。

---

## 9. 法的・免責

- 整備間隔は参考値。メーカー保証・車検とは別物
- OBD / AI 回答は参考情報。重要判断は整備工場へ
- ストア公開前に利用規約・プライバシーポリシーを用意

---

## 10. 技術方針

- 静的 Web（`index.html` / `css` / `js`）
- HTTPS または localhost（Web Bluetooth）
- BLE ELM327 系
- 将来: Flutter + 課金（Mac 環境）

---

## 11. 成功指標

- 初回3分以内に車両登録→整備1件記録
- 2台目以降の追加が迷わずできる
- OBD デモまたは実接続でメーターが動く

---

## 12. Cursor 作業ルール

1. 実装前に本SPECの「やる / やらない」を確認
2. 方針変更はコードより先に本SPECを更新
3. 車種固有ロジックを入れるときはテンプレート or ユーザー設定に閉じる
