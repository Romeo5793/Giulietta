# 家族向け — Garage Log の使い方

## URL（ブックマーク推奨）

**https://romeo5793.github.io/Giulietta/**

アプリのインストールは不要です。上の URL をブラウザで開くだけです。

---

## 端末ごとのおすすめ

| 端末 | ブラウザ | できること |
|------|----------|------------|
| Mac / Windows | **Chrome** または **Edge** | 整備手帳 + OBD 接続 |
| iPhone / iPad | **Safari** | 整備手帳のみ |
| iPhone（OBD 接続） | **Bluefy**（無料アプリ） | 整備手帳 + OBD 接続 |

OBD 接続には **BLE 対応** の ELM327 アダプタが必要です（クラシック Bluetooth のみは不可）。

---

## 初回の流れ

1. URL を開く
2. **設定** タブで自分の車を登録（ニックネーム・メーカー・車種など）
3. **ホーム** で走行距離を入力
4. **整備** タブでメニューを確認・記録を残す

---

## データについて

- 記録は **その端末のブラウザ内** に保存されます
- 家族の人同士でデータは **自動共有されません**（各自のスマホ・PCに別々に残ります）
- ブラウザのデータを消すと記録も消えます

---

## 注意

- 整備間隔・OBD 診断結果は **参考情報** です
- 重要な判断は取扱説明書・整備工場にご相談ください
